Here are some things the user should be able to do using the gcode-editor:

change tool number (from within table)
change work zone (from within table)
add timestamped comment near the top of the file
transform the gcode to the optimized form (see reference/gcode-structure-optimized) with input fields for parameters.
add any missing safety lines to beginning of code, and at beginnings and endings of N blocks
copy the text to clipboard
download the file as a .nc file type
print the table
